헤르메스 에이전트를 활용해 '미국 주식 추천주 발굴 시스템'을 구축하면, 단순히 스크리너 프로그램을 돌리는 것을 넘어 **"스스로 팩트를 체크하고 투자 논리를 제시하는 AI 애널리스트"**를 비서로 두는 것과 같습니다.
이 에이전트가 어떤 구조로 움직이고, 어떤 도구들이 필요한지 로드맵을 그려드릴게요.

### 🏗️ 미국 주식 발굴 에이전트의 4단계 워크로프 ###
유저가 *"2026년 상반기 실적 발표 기업 중, 매출 성장률이 20% 이상이면서 월가 평가가 아직 낮은 유망주 3개만 뽑아줘"*라고 명령하면 에이전트는 아래와 같이 자율적으로 움직입니다.

#### 1단계: 펀더멘탈 및 기술적 데이터 스크리닝 (Data Analysis) ####
*	행동: 에이전트가 주식 데이터 API를 호출하여 미국 증시(NYSE, NASDAQ) 전체 종목의 재무 데이터를 긁어옵니다.
*	분석: 내부 파이썬 환경에서 Pandas 라이브러리를 사용해 유저가 제시한 조건(매출 성장률 > 20%, PER 비교 등)으로 필터링하고 가치평가(Valuation) 점수를 매깁니다.

#### 2단계: 최신 뉴스 및 SEC 공시 분석 (Web Research) ####
*	행동: 필터링된 후보 기업들의 최신 미국 증권거래위원회(SEC) 공시 자료(10-Q, 10-K)와 이터, 블룸버그 등 금융 뉴스를 검색합니다.
*	이유: 숫자는 좋지만 뒤로 소송이 걸려있거나, CEO 리스크가 있는지 등 '숫자 뒤에 숨은 리스크'를 인간 애널리스트처럼 구글링으로 팩트 체크하기 위함입니다.

#### 3단계: 시장 센티먼트(여론) 체크 #### 
*	행동: 레딧(r/WallStreetBets), 트위터(X) 금융 피드, 팁랭크스(TipRanks)의 월가 애널리스트 목표주가 컨센서스를 크롤링합니다.
*	이유: 아무리 좋은 주식도 시장의 관심이 너무 없거나 반대로 거품이 낀 상태(과매수)라면 매수 타이밍으로 부적절하기 때문에 대중과 전문가의 심리를 읽는 단계입니다.

#### 4단계: 투자 추천 리포트 발행 #### 
*	행동: 수집된 모든 계량 데이터와 정성 데이터를 종합하여, 최종 추천주와 함께 **[투자 논리 / 목표가 / 리스크 요인 / 적정 진입 시점]**이 담긴 깔끔한 국문 보고서를 마크다운 형태로 출력합니다.

### 🛠️ 에이전트에게 쥐어줘야 할 필수 무기 (MCP / API) ###
맥 미니의 헤르메스 에이전트가 이 작업을 수행하려면 외부 세계와 소통할 수 있는 아래 도구(API)들을 환경설정(.env)에 연결해 주어야 합니다.
*	금융 데이터 공급원 (선택 1): * Alpha Vantage 또는 Yahoo Finance (yfinance) 플러그인 (주가 경로 및 재무제표 수집용)
*	실시간 웹 검색 도구 (선택 1): * Tavily API 또는 Perplexity API (금융 뉴스 및 SEC 공시 탐색 최적화)
*	코드 실행 환경: * 도커 내부의 Python Sandbox (에이전트가 직접 백테스팅 코드를 짜서 돌릴 수 있게 허용)

### 💡 이 에이전트가 인간보다 뛰어난 점 ###
인간 투자자는 본업이 바쁘면 관심 종목 10~20개 보기도 벅차지만, 이 에이전트는 내가 맥 미니를 켜놓고 자는 동안 밤새 미국 시장 전체(수천 개 종목)를 이 잡듯 뒤져서 아침 출근길에 완성된 보고서를 배달해 줄 수 있습니다. 말 그대로 나만의 '헤지펀드 리서치 팀'을 무료로 가동하는 셈이죠.
